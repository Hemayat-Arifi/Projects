A nicely designed Web Application using Flask framework perform as video downloader to you.

Instructions :
1. install all the necessary staff included in requirements.txt file and open a shell in the root of this project and type flask run.
2. Copy the server side ip link and paste in your browser and enjoy the app
NOTE: if you failed in step 1 try setting enviroment variable to the project app then try again step 1 --> on window: set FLASK_APP=app 

Techs & Libraries :
* Flask - as a WEB-App GUI
* Sqlite3 - Database for storing users info
* Pytube - python library for downloading youtube videos
* Flask-Login - for user authentication & authorization
* Flask-Bcrypt - for hashing user password 
* HTML5 & CSS3 - as front end techs
