Follow the steps to play the game:
1. Copy and paste all files together in same directory 
2. Run main.py first because this project is OOP based 
3. This game is multiplayer challenge game to controll the pads - use arrow keys for first player controller & use "W" and "s" keys in keyboard as second player controller
4. Ball speed increases when score goes to 3 till 7

Notes regarding Game :-
* Turtle module used for GUI
* OOP based Game
* Tidy code and readable
